# 🔐 Data Security  
## 咖米漾專業洗衣店資料安全政策

品牌：**咖米漾專業洗衣店**  
公司：**被神眷顧的股份有限公司**

---

## 📘 與隱私權政策之關係（Relationship with Privacy Policy）

本資料安全政策主要說明咖米漾專業洗衣店如何透過技術、管理與供應商安全措施，保護使用者資料與系統安全。

關於我們蒐集哪些個人資料、資料使用目的、保存期間、第三方分享情形，以及使用者依法可行使之權利，請參閱本公司之《隱私權政策》。

---

## 🛡️ 傳輸加密（Data in Transit）

所有透過咖米漾專業洗衣店電腦版網站與手機 Web View 進行的資料傳輸，均採用：

> **TLS 1.3 高強度加密技術**

TLS 1.3 是目前業界廣泛採用的安全傳輸協定，可有效降低資料在傳輸過程中遭攔截、竄改或竊取的風險，確保使用者與系統之間的通訊安全。

---

## ☁️ 雲端基礎架構（Cloud Infrastructure）

為確保服務穩定性、資料安全性與系統可用性，咖米漾專業洗衣店採用位於日本東京的雲端基礎架構。

我們的主要服務部署於：

- **Linode by Akamai**
- **Tokyo, Japan**
- **Equinix Data Center Service Provider**

此外，我們亦採用 **Cloudflare R2 Object Storage** 作為備份資料與物件儲存服務。

我們所採用或參考之雲端與基礎架構服務供應商，包括 **Linode by Akamai**、**Amazon Web Services (AWS)** 及 **Cloudflare**，均具備多項國際安全與隱私管理認證，包括：

- **SOC 2**
- **ISO/IEC 27001**
- **ISO/IEC 27017**
- **ISO/IEC 27018**
- **ISO/IEC 27701**

上述認證涵蓋資訊安全管理、雲端服務安全、個人資料保護、雲端隱私保護與隱私資訊管理等面向，有助於確保我們採用的基礎架構與雲端服務符合國際資安與隱私保護標準。

---

## 💾 資料備份與物件儲存（Backup & Object Storage）

為降低資料遺失風險並提升災難復原能力，咖米漾專業洗衣店定期進行資料備份。

備份資料儲存於：

- **Cloudflare R2 Object Storage**
- **Tokyo, Japan**

Cloudflare R2 作為雲端物件儲存服務，用於保存系統備份資料，協助我們在系統異常、設備故障或不可預期事件發生時，能夠進行資料還原與服務恢復。

---

## 🧩 備份與復原管理（Backup & Recovery Management）

我們定期執行系統與資料備份，並將備份資料儲存於獨立雲端物件儲存服務，以降低系統異常、設備故障或人為錯誤造成之資料遺失風險。

備份資料依服務營運與系統復原需求進行保存，並透過適當權限控管保護，避免未授權存取。

---

## 🌏 災難復原與備援設計（Disaster Recovery & Redundancy）

為提升服務可靠性，我們採取以下措施：

- 定期備份系統與用戶資料
- 將備份資料儲存於獨立物件儲存服務
- 主要服務與備份儲存皆位於日本東京區域
- 定期檢視備份與復原流程

### 主要服務位置

- 🇯🇵 **日本東京（Tokyo, Japan）**

### 備份儲存位置

- 🇯🇵 **日本東京（Tokyo, Japan）**

此設計可降低單一系統故障造成的資料遺失風險，並協助咖米漾專業洗衣店在異常情況下盡快恢復服務。

---

## 🔑 存取權限控管（Access Control）

咖米漾專業洗衣店採取權限分級管理原則，僅授權必要人員於執行工作所需範圍內存取相關系統與資料。

所有系統管理、訂單處理與客服作業，皆依照最小權限原則進行控管，以降低未授權存取、資料誤用或人為疏失之風險。

---

## 🖥️ 系統安全維護（System Security Maintenance）

為維持系統安全與服務穩定，我們會依實際營運需求進行系統維護、安全更新與設定檢視。

相關維護作業包括但不限於：

- 系統與套件安全更新
- 伺服器設定檢視
- 存取權限檢查
- 備份流程檢視
- 異常狀況排查與修復

---

## 🚨 資安事件應變（Security Incident Response）

若發生可能影響使用者資料安全或系統穩定性之事件，咖米漾專業洗衣店將依事件嚴重程度進行調查、控制、修復與後續改善。

如事件依法須通知主管機關或受影響使用者，我們將依適用法令規定辦理通知。

---

## 🤝 第三方服務安全管理（Third-Party Security Management）

我們僅選擇具備可信賴安全管理能力之第三方服務供應商，並依服務必要性與資料處理目的使用相關服務。

對於涉及資料處理或系統營運之服務供應商，我們會定期檢視其安全性、服務穩定性與資料保護措施。
