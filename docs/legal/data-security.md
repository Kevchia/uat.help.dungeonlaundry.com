# 🔐 Data Security  
## 地下城洗衣店資料安全政策

---

## 🛡️ 傳輸加密（Data in Transit）

所有透過地下城洗衣店之電腦版與手機 Web View 進行的資料傳輸，均採用：

> **TLS 1.3 高強度加密技術**

此技術為全球金融機構與國際信用卡交易所使用之標準安全協定，確保資料在傳輸過程中不被攔截或竄改。

---

## ☁️ 雲端基礎架構（Cloud Infrastructure）

為確保高可用性與資料安全性，我們採用全球領先的雲端服務平台：

- **Amazon Web Services (AWS)**
- **Google Cloud Platform**

所有用戶資料皆經加密後儲存於上述雲端環境中。

---

## 🌏 災難復原與跨區備援（Disaster Recovery & Redundancy）

為防範自然災害或不可抗力事件，我們：

- 每日定時備份所有用戶資料
- 採用跨洲異地儲存架構

### 備份儲存位置

- 🇯🇵 日本東京（Tokyo, Japan）
- 🇺🇸 美國奧勒岡（Oregon, USA）

此設計確保在單一區域發生重大事故時，服務仍可快速恢復。

## 🔎 安全承諾

地下城洗衣店致力於：

- 採用業界標準加密技術
